# E-Commerce Sales Analysis Dashboard (Power BI)

## Project Overview
This project analyzes an e-commerce dataset using Power BI to identify sales trends, customer segments, and product performance.

## Tools Used
- Power BI
- Excel / CSV dataset
- Data visualization techniques

## Dashboard Features
- KPI cards for Total Sales, Total Profit, and Quantity Sold
- Sales by Category analysis
- Profit by Category comparison
- Regional sales performance
- Top selling products
- Discount vs Profit relationship
- Customer segment distribution
- Interactive filters (Region and Category)

## Insights
- Furniture category generated the highest revenue.
- Central region showed the highest sales.
- Consumer segment contributes the largest share of customers.
